# OpenClaw

AI-driven automation framework for RF MEMS (BAW/SAW) filter design.

## Vision
Bridge the gap between PDK development, layout automation, and AI-assisted 
filter design for fabless MEMS product development.

## Current Status (WIP)
- [x] FBAR PCell geometry engine (L10/L20/L50 validated)
- [x] DoE table generator
- [ ] EM co-simulation interface
- [ ] AI optimization loop
